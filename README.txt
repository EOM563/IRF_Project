# IRF_Project

A porgram egy kisvállalat készleteinek lekérdezését segíti adatbázisból.
Ehhez gombok és labelök segítségével vizualizálja a készleteket, valamint ezen gombok segítségével kiválasztható,
hogy mely termékeket, termék csoportokat, szeretnék egy formázott kimutatásban, exceles formában kinyerni.
