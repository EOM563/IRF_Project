# IRF_Project

A program egy kisvállalat készleteinek lekérdezését segíti adatbázisból.
Ehhez gombok és labelök segítségével vizualizálja a készleteket, valamint ezen gombok segítségével kiválasztható,
hogy mely termékeket, termék csoportokat, fogja egy formázott kimutatásban, exceles formában kinyerni. 

Vizualizáció:
-> Adatbázis alappján termék kategóriánként UserControlokat generál és ezeken jelennek meg a termékek gombjai
-> A gombok segítségével kiválasztható, hogy mely termékeket szeretnénk az excelben látni
-> Labelökön megjelennek az egyéb fontos készletinformációk
-> Gombnyomásra a kijelölt elemek megjelennek excelben, formázottan
